MinPriorityQueue

Supports Debug ModeSupports __lenSupports __tostring

In a min priority queue, elements are inserted by the priority in descending order and the maximum value is always removed first from the queue.

This uses a sorted array internally for the maximum speed possible when inserting, popping, and removing by priority.
