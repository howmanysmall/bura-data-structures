SimpleTrie

Supports Debug ModeSupports __tostring

The SimpleTrie class implements a basic trie (prefix tree) data structure for efficient storage and retrieval of strings. It allows you to insert words, search for exact matches, and check if any words in the trie start with a given prefix. If you need utf8 support, use the Utf8Trie class.

Features:

  • Efficient String Storage: Stores strings in a hierarchical structure for fast retrieval and minimal redundancy.
  • Word Insertion: Add words to the trie using the Insert method.
  • Exact Word Search: Determine if a word exists in the trie with the Search method.
  • Prefix Matching: Use the StartsWith method to check if any words start with a given prefix.
  • Supports Debug Mode: Enable debug mode by setting _G.__DEV__ = true before requiring the module to enforce type checks and provide detailed error messages.

Example Usage:

_G.__DEV__ = RunService:IsStudio()
local DataStructures = require("DataStructures")

-- Create a new trie
local trie = DataStructures.SimpleTrie.new()

-- Insert words
trie:Insert("apple")
trie:Insert("app")
trie:Insert("banana")

-- Search for exact words
print(trie:Search("apple"))   -- Output: true
print(trie:Search("app"))     -- Output: true
print(trie:Search("apples"))  -- Output: false

-- Check for prefixes
print(trie:StartsWith("app"))   -- Output: true
print(trie:StartsWith("ban"))   -- Output: true
print(trie:StartsWith("bat"))   -- Output: false

-- Check if an object is a SimpleTrie
print(DataStructures.SimpleTrie.Is(trie))     -- Output: true
print(DataStructures.SimpleTrie.Is({}))       -- Output: false

-- String representation
print(tostring(trie))          -- Output: "SimpleTrie"

    • Creates a new SimpleTrie instance.
    • Returns: A new SimpleTrie object.

  • SimpleTrie:Insert(word: string): void

    • Inserts a word into the trie.
    • Parameters:
      • word (string): The word to insert into the trie.
    • Errors:
      • DebugTypeError: Thrown when debug mode is enabled and word is not a string.

  • SimpleTrie:Search(word: string): boolean

    • Searches for an exact word in the trie.
    • Parameters:
      • word (string): The word to search for.
    • Returns: true if the word exists in the trie, false otherwise.
    • Errors:
      • DebugTypeError: Thrown when debug mode is enabled and word is not a string.

  • SimpleTrie:StartsWith(prefix: string): boolean

    • Checks if any word in the trie starts with the given prefix.
    • Parameters:
      • prefix (string): The prefix to check.
    • Returns: true if any word starts with the prefix, false otherwise.
    • Errors:
      • DebugTypeError: Thrown when debug mode is enabled and prefix is not a string.

  • SimpleTrie.Is(value: any): boolean

    • Checks if the given value is an instance of SimpleTrie.
    • Parameters:
      • value (any): The value to check.
    • Returns: true if value is a SimpleTrie instance, false otherwise.

  • SimpleTrie.instanceof(value: any): boolean

    • Alias for SimpleTrie.Is.
