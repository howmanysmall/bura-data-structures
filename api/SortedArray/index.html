SortedArray

OptimizationSupports Debug ModeSupports __lenSupports __tostring

A class to create automatically sorted arrays. Must contain objects comparable to one another (that can use the < and == operators). Numbers and strings support these operators by default.

baseArray?: {T},--

An array of data which will be sorted upon instantiation. If this is omitted, an empty array is used.

comparison?: ComparisonFunction<T>--

An optional comparison function which is used to customize the element sorting, which will be given two elements a and b from the array as parameters. The function should return a boolean value specifying whether the first argument should be before the second argument in the sequence. If no comparison function is passed, the Luau-default a < b sorting is used.
